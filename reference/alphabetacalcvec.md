# Vector-based - Calculate Alpha and Beta from Mean and Variance

Vector-based - Calculate Alpha and Beta from Mean and Variance

## Usage

``` r
alphabetacalcvec(mu, var)
```

## Arguments

- mu:

  Vector of mean.

- var:

  Vector of variance.

## Value

Numeric matrix of alpha and beta.

## Examples

``` r
abc <- alphabetacalcvec(c(0.5, 0.5), c(0.01, 0.01))
```
